## Prompt 1

### Actúa como si fueses el dueño de la página y quisieses añadir funcionalidades a la página para que sea mas util o mejore las visitas.

Funciona por que actúa con un rol en el que busca solucionar problemas y mejorar la calidad de la página, da ideas de mejora de utilidad de la página e ideas que le hacen pensar si el cliente volvería o no con esa implementación a la página.

## Prompt2

### Actúa como un ingeniero de software senior con 10 años de experiencia en JavaScript. Revisa este código buscando errores lógicos, casos de borde no cubiertos y posibles mejoras para un código mas legible y seguro.

Funciona mejor que decirle directamente que encuentre errores en tu código por que asume un rol en el que intenta dejar el código con unos estándares de calidad mas altos y practicas de código mas claras y seguras en vez de darte la respuesta mas rápida y simple.

## Prompt3

### Explica como funciona el siguiente código(adjuntas código) como si lo explicaras a alguien un desarrollador junior.

Funciona por que te permite entender de forma mucho mas clara e intuitiva código que no es claro o con el que no estas tan familiarizado, también sirve cuando has pedido a la ia crear una función que haga algo en específico, que no has usado o no sabes bien como funciona, es útil que te lo explique de forma mas intuitiva para luego tu poder aplicar tu función en el proyecto de la manera que necesites.

## Prompt4

### Corrige como si fueses un profesor de ortografía todas las faltas que haya en el siguiente texto.

No es muy util para la programación pero en mi caso ha sido util para el README y el resto de archivos .md quede sin ninguna falta por si alguna se ha escapado, asi como escribir correctamente comentarios.

## Prompt5

Funciona por que la ia tiene gran capacidad estética y puede aplicar un aspecto visual moderno o en relación a tu código sin necesidad de abrir css desde cero.

### Actúa como un experto en ciberseguridad analiza el código en busca de posibles amenazas como inyección de SQL o XSS, documenta las posibles vulnerabilidades y sugiere como solucionarlas.

La IA ha sido entrenada con bases de datos de vulnerabilidades comunes, lo que le permite identificar patrones de código inseguro que un humano podría ignorar.

### Genera un archivo CSS en el que estén los botones del siguiente código HTML, usa un estilo minimalista y que tenga un efecto :hover.

Funciona por que al ver tu código HTML hace el CSS con tus clases e id's a demás de darle un estilo concreto que le has determinado de manera rápida, sencilla y que puedes modificar posteriormente pero en menos tiempo y de forma profesional.
